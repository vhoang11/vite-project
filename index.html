Van's HTML Reference Page

main

Contains the main content of the web page

        
          <main>
            This is the main content
            <p>This is a tag in the main tag</p>
          </main>

p

Creates a paragraph for the web page

        
          <p>
            This is the p content
            <p>This is a p tag in a p tag</p>
          </p>

bold

This will make anything in between the tag bold

        
          <b>
            This is the bold content
            <p>This text would be bold</p>
          </b>
